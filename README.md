# DIP轻量工具包

专为社区医院设计的离线版DIP（按病种分值付费）辅助工具包，帮助医务人员在DIP过渡期间快速适应新政策，提高工作效率和准确性。

## 功能特点

### 1. 预分组速查表
- 快速查询常见病种的DIP分组规则
- 支持按科室、严重程度筛选
- 模糊搜索疾病名称、ICD编码、DIP编码
- 查看详细分组信息和注意事项

### 2. 病案首页自检清单
- 系统化检查病案首页填写质量
- 15项关键检查点，涵盖基本信息、诊断、手术、费用等
- 实时进度跟踪和质量评分
- 一键生成检查报告

### 3. 费用偏离预警计算器
- 计算实际费用与标准费用的偏离度
- 根据医院等级自动调整预警标准
- 提供分级预警（正常/关注/高风险）
- 给出具体改进建议

## 技术特点

- **完全离线可用**：所有数据存储在本地浏览器中，无需网络连接
- **响应式设计**：适配电脑、平板、手机等多种设备
- **数据安全**：所有数据本地存储，不会上传到任何服务器
- **易于使用**：直观的用户界面，无需复杂培训

## 使用方法

### 快速开始
1. 下载本工具包到本地计算机
2. 用浏览器打开 `index.html` 文件
3. 无需安装，直接开始使用

### 数据管理
- **导出数据**：点击页脚"导出数据"按钮，可将所有数据导出为JSON文件备份
- **导入数据**：可通过JavaScript控制台调用 `importFromJSON()` 函数导入数据
- **重置数据**：点击页脚"重置数据"按钮可清除所有本地数据

### 离线使用说明
本工具包设计为完全离线使用：
- 首次使用时自动加载示例数据
- 所有操作记录保存在浏览器本地存储中
- 即使关闭浏览器，数据也不会丢失（除非清除浏览器数据）
